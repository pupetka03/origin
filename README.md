## 📦 1. Типи Даних

Мова підтримує два вбудованих типи:

- **`int`** — представляє будь-які числа (цілі та дробові).
- **`str`** — представляє рядки (текст).

### Оголошення Змінних

Змінні оголошуються з обов'язковим зазначенням типу. **Крапка з комою (`;`)** в кінці кожного рядка є обов'язковою.

```
int x = 10;
str y = "hello";
int pi = 3.1415;
```

### Динамічна Зміна Типу (Re-typing)

Змінній можна надати значення іншого типу, але ключове слово типу потрібно писати завжди явно, навіть якщо змінна вже була оголошена раніше.

```
int x = 10;
str x = "hello";   // ✅ Правильно: тип вказано знову

// x = "hello";    // ❌ ПОМИЛКА: тип не вказано!
```

> ⚠️ **Важливо:** Просте переприсвоєння без типу (`x = "hello";`) заборонено.

## 🖨️ 2. Виведення Даних (`print`)

Функція `print` використовується для виведення значень у консоль.

### Базовий вивід

```
print("Hello, World!");
```

### Пробіли між значеннями

Коми в `print` заборонені. Для додавання пробілів між частинами використовуйте зворотну косу риску `\`.

```
str name = "Artem";
print("Hello" \ name);        // Виведе: Hello Artem
print("Hello" \ \ name);      // Виведе: Hello  Artem (два пробіли)
print(x \ "текст" \ y);       // Можна комбінувати змінні та рядки
```

---

## 🧮 3. Вирази Всередині `print`

Щоб обчислити вираз безпосередньо всередині `print`, використовуйте фігурні дужки `{}`. Вираз всередині обчислюється, а результат одразу виводиться.

```
int a = 5;
print({a + 5});                    // Виведе: 10
print("Результат:" \ {a * 2});     // Виведе: Результат: 10
```

## 🔍 4. Функції та Інтроспекція

### Отримання типу змінної — `type()`

Вбудована функція `type()` повертає тип змінної у вигляді рядка.

```
int value = 42;
str valueType = (type(value));
print("Тип змінної:" \ valueType);    // Виведе: Тип змінної: int
```

### Зчитування вводу — `scan()`

Функція `scan()` зчитує введення від користувача. Тип вказується в дужках.

```
int age = (scan(int));     // Чекає на введення числа
str name = (scan(str));    // Чекає на введення рядка
```

> ⚠️ Якщо користувач введе не число при `scan(int)` — виникне помилка.

## 🔁 5. Цикли (`for`)

### Синтаксис

```
for <ініціалізація>, <умова>, <крок>,
    <тіло>
end;
```

- Кома `,` розділяє частини циклу.
- Тип змінної вказується щоразу в ініціалізації та кроці.
- Цикл обов'язково закінчується `end;`.

### Приклад

```
int limit = 5;

for int i = 0, i < limit, int i = {i + 1},
    print("Ітерація:" \ {i});
end;
```

| Частина | Приклад | Опис |
|---|---|---|
| Ініціалізація | `int i = 0` | Створення лічильника |
| Умова | `i < limit` | Поки виконується — цикл іде |
| Крок | `int i = {i + 1}` | Оновлення після кожної ітерації |
| Тіло | `print(...)` | Інструкції (можна кілька через `;`) |

### Одностроковий `for`

```
for int i = 0, i < 10, int i = {i + 1}, print({i}); end;
```

### Вкладені `for`

```
for int i = 0, i < 3, int i = {i + 1},
    for int y = 0, y < 3, int y = {y + 1},
        print({i} \ {y});
    end;
end;
```

## 📜 6. Зведення Правил (Cheat Sheet)

| | Правило |
|---|---|
| ❗ | Кожна інструкція закінчується `;` |
| ❗ | Тип змінної вказується **завжди** — при оголошенні та переприсвоєнні |
| ℹ️ | `int` працює для всіх чисел (як `5`, так і `3.14`) |
| ℹ️ | `{}` всередині `print` — обчислення виразу |
| ℹ️ | Пробіли в `print` контролюються символом `\`. Коми заборонені |
| ℹ️ | Цикл `for` вимагає явного типу в ініціалізації та кроці |
| ℹ️ | Вкладені `for` — підтримуються, кожен закривається своїм `end;` |

## 🚀 7. Приклад Програми

```
// Оголошення змінних
int counter = 0;
str message = "Лічильник:";

// Цикл для підрахунку
for int i = 1, i < 4, int i = {i + 1},

    // Оновлюємо лічильник (переприсвоєння з типом!)
    int counter = {counter + i};

    // Виводимо результат
    print(message \ {counter} \ "після додавання" \ {i});

end;

// Перевірка типу
str finalType = (type(counter));
print("Фінальний тип counter:" \ finalType);
```

**Вивід:**
```
Лічильник: 1 після додавання 1
Лічильник: 3 після додавання 2
Лічильник: 6 після додавання 3
Фінальний тип counter: int
```
